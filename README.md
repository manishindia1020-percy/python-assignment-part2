# Restaurant Order Management System (Part 2)

This project uses Python data structures like dictionaries and lists to simulate a restaurant system.

## What I implemented

### Task 1 — Menu Exploration

* Displayed menu grouped by category
* Counted total and available items
* Found most expensive item
* Filtered items under ₹150

### Task 2 — Cart Operations

* Added items to cart with quantity handling
* Prevented adding unavailable or invalid items
* Removed and updated cart items
* Generated order summary with GST calculation

### Task 3 — Inventory Tracking

* Used deep copy to create backup
* Updated stock after order
* Displayed reorder alerts

### Task 4 — Sales Analysis

* Calculated revenue per day
* Found best-selling day
* Identified most ordered item
* Printed all orders using enumerate

## How to run

```
python part2_order_system.py
```

## Notes

I avoided complex shortcuts and tried to simulate how an actual system would behave step by step.
